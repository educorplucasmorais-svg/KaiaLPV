# KaiaLPV

Página de venda da Kaia - Landing page moderna e responsiva.

## Descrição

Esta é a landing page oficial da Kaia, uma plataforma inteligente que oferece soluções inovadoras e práticas para transformar seu dia a dia.

## Estrutura do Projeto

- `index.html` - Página principal com estrutura HTML semântica
- `styles.css` - Estilos CSS com design moderno e responsivo

## Recursos

- Design responsivo para todos os dispositivos
- Navegação suave com scroll animado
- Seções: Hero, Recursos, Sobre, Contato
- Formulário de contato integrado
- Gradientes e animações modernas

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/kaia-landing.git
   ```

2. Abra o arquivo `index.html` no navegador

3. Ou utilize um servidor local:
   ```bash
   npx serve .
   ```

## Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animações)
